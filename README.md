# Sicegah

Ini adalah repository iot Sicegah 

Sicegah adalah aplikasi inovatif yang menggabungkan teknologi Internet of Things (IoT), Artificial Intelligence (AI), dan pemetaan geospasial untuk memberikan solusi deteksi dini dan pencegahan penyakit menular. Dengan memanfaatkan wearable device serta input manual, aplikasi ini menganalisis potensi penyakit menggunakan AI dan memberikan peringatan dini yang akurat. Pendekatan proaktif ini membantu masyarakat melakukan tindakan pencegahan lebih cepat, sekaligus meningkatkan kesadaran akan kesehatan publik.

## Installation

Clone the github repo to your directory

```bash
git clone https://github.com/KSM-IoT-UPNVJ/Sicegah.git
```

## Usage

```python
# first time run

npm install

# run
npm run dev
```
# Framework

- React + Vite untuk membuat frontend aplikasi
- React Leaflet untuk membuat pengelompokan wilayah 
- postgreSQL untuk membuat database aplikasi
